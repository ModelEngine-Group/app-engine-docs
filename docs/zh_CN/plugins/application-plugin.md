# 应用插件

应用插件模块用于管理用户自主开发的插件，支持**创建、编辑和删除**插件内容。插件可在应用或工作流中调用，扩展模型能力或集成外部系统，适用于需要个性化处理逻辑或增强功能的场景。

## 功能说明

- **创建插件**：上传自定义开发的插件包
- **编辑插件**：修改已有插件的配置信息
- **删除插件**：移除不再使用的插件

## 使用场景

- 扩展大模型能力（如天气查询、翻译、图像识别等）
- 集成外部系统API
- 实现自定义数据处理逻辑
- 增强工作流节点的功能
