# 循环节点

## 节点说明

循环节点用于对指定的数组进行循环操作，对数组中的每一条数据执行指定的工具。

## 使用方式

* 在"工具"模块中，选择需要循环调用的工具。
* 当选择工具后，其参数会自动显示在"输入模块"中，可以选择其中一个字段名作为循环的配置。
* 被选择的字段应配置为数组类型，数组中的每个元素将作为一次循环的入参传入工具中。

## 输出说明

* 输出为一个名为 `output` 的数组，包含每次工具调用的结果，顺序与输入数组顺序一致，每次调用的返回结果依次写入其中。

## 错误处理

* 若某次工具调用失败，循环节点将直接结束，并返回该次调用的错误信息。

## 注意事项

* 用户需确保用于循环的数组中的元素类型与所选工具的参数类型一致，否则将导致调用报错。
* 循环节点适用于批量处理、文件解析、数据提取等需要对多个相似对象重复操作的场景。
