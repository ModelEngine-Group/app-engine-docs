# 智能编排表单节点

## 节点说明

智能编排表单节点用于在AI工作流中动态生成表单，支持灵活地配置表单项以及表单项的数据来源和输出。

## 配置智能表单

智能表单节点包括以下两个模块：表单项和输出。

### 1. 表单项模块

点击表单项边上的加号，可以增加表单项，每个表单项都可以独立设计，包括以下配置：

| 配置项     | 说明                                      |
| ------- | --------------------------------------- |
| 表单项名称   | 此表单项的唯一标识符，用于后续调用和引用                    |
| 表单项展示名称 | 显示在实际表单上的名称                             |
| 表单项渲染组件 | 选择使用的表单输入组件                             |
| 表单项类型   | 输入的类型（String, Integer, Number, Boolean） |
| 表单项默认值  | 表单项的默认值，可直接输入或引用前置节点内容                  |

### 表单项渲染组件说明

以下为可选的表单渲染组件及适用类型：

| 组件类型     | 支持类型                    | 额外说明                               |
| -------- | ----------------------- |------------------------------------|
| Input    | String, Integer, Number | 普通输入框，用于简单文本或数字输入                  |
| Switch   | Boolean                 | 开关组件，用于布尔类型选择（开/关）                 |
| Radio    | String, Integer, Number | 单选按钮，需提供与类型一致的选项列表 **（参数必须为列表格式）** |
| Select   | String, Integer, Number | 下拉选择框，需提供与类型一致的选项列表 **（参数必须为列表格式）**  |
| CheckBox | 无                       | 复选框，需提供一个选项列表，可多选 **（参数必须为列表格式）**    |
| Label    | String, Integer, Number | 显示静态标签信息，不可输入                      |

### 2. 输出模块

输出模块定义表单提交后的数据格式，每增加一个表单项，`output`对象中将新增一个对应的字段。
