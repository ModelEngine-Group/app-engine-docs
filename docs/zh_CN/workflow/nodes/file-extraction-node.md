# 文件提取节点

## 节点说明

文件提取节点用于提取用户在对话界面中上传的文件内容，支持多种文档格式，适用于需要对用户上传文件内容进行分析、处理或存储的业务场景。

## 节点功能

* 支持提取上传文件中的文本内容，输出为 `String` 类型。
* 用户可通过对话界面上传文件，系统将自动记录文件地址。
* 输入字段需配置为引用 `系统上下文 > fileUrls`，用于获取用户上传文件的地址。
* 支持上传 pdf、txt、docx、xlsx、md、htm、html、csv 等文件格式。
* 单个文件最大 15MB，最多可上传 5 个文件。

## 模块结构

### 1. 输入模块

* 字段名：`files`
* 字段值：引用 `系统上下文 > fileUrls`

### 2. 输出模块

* 输出变量名为 `output`，类型为 `String`
* 包含从文件中提取出的纯文本内容

---

文件提取节点简化了对上传文件内容的处理流程，是文档类任务中非常实用的节点组件。
