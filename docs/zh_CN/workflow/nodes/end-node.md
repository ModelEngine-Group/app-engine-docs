# 结束节点

## 节点说明

结束节点用于AI工作流的终点，可选择直接输出结果或使用智能表单展示结果，支持灵活输出配置。

## 模式选择

结束节点包含两种模式：

* **直接输出结果**
* **智能表单展示结果**（目前仅支持人工表单）

### 1. 直接输出结果

#### 输出结果至聊天

* 如果前置大模型节点已选择了输出结果至聊天，结束节点可以选择不勾选此选项，即不再额外输出到聊天界面。
* 如果需要将结果展示在聊天界面，则勾选此选项。

#### 配置输出内容

结束节点的输出为一个 `finalOutput` 对象，可根据需求自由添加输出的值。

* 点击输出右边的加号按钮添加新的输出字段。
* 若添加多个输出字段，每个字段将另起一行。
* **注意：当添加两个及以上的值时，将影响流式输出的效果。**

![直接输出结果](../assets/images/结束节点-直接输出结果.png)

### 2. 智能表单展示结果

* 目前仅支持人工表单，选择该模式后，工作流的最终结果会以智能表单的形式展示给用户。

![智能表单展示结果](../assets/images/结束节点-智能表单展示结果.png)

## 常见问题

1. **什么时候需要选择输出结果至聊天？**

  * 当需要最终工作流结果直接呈现在用户的聊天界面时勾选。

2. **如何添加多个输出字段？**

  * 点击输出模块右侧的加号按钮，逐个添加需要的字段，注意多个字段可能会影响流式输出。

3. **选择智能表单展示结果有哪些限制？**

  * 当前仅支持人工表单模式，其他类型智能表单暂未支持。
