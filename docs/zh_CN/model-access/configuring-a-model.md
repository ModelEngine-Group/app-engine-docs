# 快速开始：配置你的大模型服务

欢迎使用 **模型配置应用**，本应用帮助你快速完成模型的添加、默认模型切换、API Key 绑定等操作，适用于需要集成多种大模型的业务场景。

## 应用原理

本应用通过可视化工作流封装了以下关键能力：

- 支持添加自定义模型信息（模型名称、baseUrl 、API Key ）；
- 支持设置默认模型，供后续对话调用使用；
- 支持删除模型、退出模型管理；
- 所有配置将自动同步并可供平台或调用方使用；

你只需回答几个简单问题，系统会自动完成模型配置流程。

## 使用步骤

### 第一步：查看模型配置

在平台首页应用市场中搜索 模型配置应用，点击**创意灵感**中的**开始配置**，或直接发起对话。

### 第二步：添加模型

点击添加模型并填写模型基本信息：

- **模型名称**：建议填写简洁可识别的名称，如 Qwen-7B-Chat；
- **Base URL**：模型的服务接口地址；
- **API Key**：用于调用该模型的密钥；

填写完成后，点击**确认**按钮，即可将模型保存至系统。

### 第三步：设置默认模型（可选）

在模型列表中，点击目标模型的操作菜单，选择**设为默认模型**，系统将优先使用该模型进行对话与推理任务。

### 第四步：删除模型

在模型列表中，点击目标模型的操作菜单，选择**删除模型**，系统将删除目标模型。
