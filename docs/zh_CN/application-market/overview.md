# 应用市场

用户在应用市场中可以看到已发布的应用并进行应用的对话。

## 前提条件

应用发布成功。

## 操作步骤

1.  在ModelEngine导航栏，选择"应用使能 > 市场 > 应用市场"。

    进入"应用市场"界面。

2.  单击某个应用，进入该应用的对话界面。

    >**说明：**
    >也可以通过选择"应用使能 > 应用开发 > 应用开发"进入"我的应用"界面，单击某个已发布的应用，再单击"去编排"，在界面右侧进行应用对话。

3.  在对话栏输入对话信息展开应用对话。应用对话相关参数详见表1。

    **表 1**  应用对话参数

    | 参数 | 说明 |
    | --- | --- |
    | 更新日志 | 应用的更新日志。单击通知图标可查看更新日志。 |
    | 历史聊天 | 应用的历史聊天记录。单击历史记录图标查看历史聊天。<ul><li>单击"清空"可将历史聊天记录全部清除。</li><li>单击"继续聊天"可再次发送该对话信息。</li><li>单击右侧菜单图标再单击删除，可删除该对话信息。</li></ul> |
    | 多轮对话 | 多轮对话默认开启，单击开关可关闭此功能。 |
    | 创意灵感 | 单击灯泡图标可显示已创建的创意灵感。 |
    | 对话配置 | 在开始节点的Question参数中勾选"在对话配置显示"，在聊天页面的对话配置中显示，用户可输入设置问题。单击配置图标可查看对话配置。 |
    | 猜你想问 | 首次与应用对话时的推荐问题，最多创建3个，创建好的问题可在进行应用对话时单击使用。 |
