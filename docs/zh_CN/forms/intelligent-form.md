# 智能表单

用户可以对表单进行创建、编辑以及删除等操作。本章节支持用户通过react、vue或原生html的形式开发智能表单并上传，应用开发者可以根据需求自定义表单，并通过编排的方式使用智能表单节点。这样用户不仅可以在对话中与流程中间过程进行交互，还可以以表单形式查看流程最终结果。

### 前提条件

使用Chrome浏览器的无痕模式登录ModelEngine环境，需要在设置中把"阻止第三方cookie"功能关闭，否则智能表单无法正常显示。

### 创建表单

#### 操作步骤

- 步骤1 在ModelEngine导航栏，选择"应用使能>应用开发>智能表单"

  进入"智能表单"界面。

- 步骤2  单击"创建"

  进入"创建表单"界面。

- 步骤3  配置表单的相关参数，参数的详细信息如表1 所示。
  表1 创建表单

  | 参数               | 说明                                                         |
    | ------------------ | ------------------------------------------------------------ |
  | 名称               | 表单的名称。                                                 |
  | 描述               | 表单的描述信息。                                             |
  | 上传智能表单组件包 | 上传智能表单组件包。<br />1. 单击"使用帮助"，查看创建表单组件的详细说明<br/>2. 单击"下载模板表单"，将模板下载至本地。<br/>3. 上传组件包文件。<br/>    -  单击"将文件拖到这里，或者单击上传"区域，在弹出的对话框中选择本地已准备好的组件包并单击"打开"。<br/>    -  直接将插件文件拖到此区域中。 |
  | 组件预览图         | 智能表单组件包上传成功之后的预览图。                         |

- 步骤4  单击"确定"，成功创建表单。

  如需取消创建，单击"取消"。

### 编辑表单

#### 前提条件

表单已经成功创建。

#### 操作步骤

- 步骤1  在ModelEngine导航栏，选择"应用使能>应用开发>智能表单"进入"智能表单"界面。

- 步骤2 单击需要编辑的表单右侧操作栏下方的"编辑"。

  进入"编辑表单"界面。

- 步骤3 根据表1 编辑表单的相关信息。

- 步骤4 编辑完成后单击"确定"。

  如需取消编辑，单击"取消"。

### 删除表单

#### 前提条件

表单已经成功创建。

#### 操作步骤

- 步骤1 在ModelEngine导航栏，选择"应用使能>应用开发>智能表单"进入"智能表单"界面。

- 步骤2 单击需要删除的表单右侧操作栏下方的"删除"弹出"确定删除"的对话框。

- 步骤3 单击"确定"，成功删除表单。

  如需取消删除，单击"取消"

> 注意： 删除后会导致正在运行的应用不可用，请谨慎操作。
