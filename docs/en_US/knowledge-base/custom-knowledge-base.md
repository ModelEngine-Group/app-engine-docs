# Custom Knowledge Base

## Overview

Custom knowledge bases allow integration of external knowledge sources to enhance AI model capabilities. This guide focuses on integrating Baidu Qianfan knowledge base services.

## Prerequisites

- Baidu Qianfan account
- API key for knowledge base access
- Basic understanding of knowledge base concepts

## Integration Steps

### 1. Account Setup
- Create a Baidu Qianfan account
- Generate API credentials
- Configure access permissions

### 2. Knowledge Base Configuration
- Define knowledge base structure
- Upload or connect data sources
- Configure search and retrieval settings

### 3. Integration with ModelEngine
- Add knowledge base connection details
- Configure query parameters
- Test integration functionality

## Benefits

- Enhanced AI model knowledge
- Domain-specific information access
- Improved response accuracy
- Scalable knowledge management
