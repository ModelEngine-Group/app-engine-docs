# Custom Model

**——Flexible Integration · One-Click Deployment · Full-Process Management**

Provides **standardized API interface management capabilities** for developers and enterprises. Through simple `API Key + Base URL` configuration, quickly integrate third-party or private models, achieving **zero infrastructure maintenance** AI capability integration.

---

Open the `Model Configuration Application` in the application market to configure models.

![Configure Model](../assets/images/配置模型.png)

Supports users to fill in the **Base URL** and **API Key** of any model, completing the integration of mainstream AI services (such as OpenAI, Gemini, Claude, or privately deployed models) in seconds.

Automatically verifies interface connectivity, generates standardized call templates, and supports selecting custom models in the `Foundation Model Node` in other workflow orchestrations.
